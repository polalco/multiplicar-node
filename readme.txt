Utilizamos npm para descargar archivos y modulos necesarios

Instalar paquetes que se usan globalmente
npm install -g 'nombre-paquete'

Instalar local para el proyecto
npm install --save 'nombre-paquete'

Instalar local herramientas o dependencias de desarrollo
npm install --save-dev 'nombre-paquete'

Desinstalar un paquete instalado.
npm uninstall 'nombre-paquete'
